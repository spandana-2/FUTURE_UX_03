# FUTURE_UX_03
CRM Dashboard UI
The Goal
A simple, one-page tool for agencies to manage new leads and daily tasks.

Features
Lead Pipeline: Track potential deals through different stages.

Quick Stats: Instant view of pipeline value and active clients.

Task Manager: A built-in checklist for client follow-ups.

Design Logic
Efficiency: Sidebar navigation for fast switching between views.

Clarity: Color-coded tags to show lead status at a glance.

Focus: Minimalist layout to reduce distractions.

Usage
Open index.html in your browser.
